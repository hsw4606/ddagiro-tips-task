# 00. 프로젝트 개요 — 딸기로컴퍼니 TIPS 과제

## 배경
**딸기로컴퍼니**는 프리미엄 디저트 전문 배송·중개 서비스다(https://ddalgiro.com/). 디저트 제조 입점 업체와 소비자를 연결하는 플랫폼 사업 구조를 가진다.

본 프로젝트는 딸기로컴퍼니가 **TIPS(민간투자 주도형 기술창업 지원, 정부 R&D 프로그램)** 과제로 채택되기 위해, 제안할 기술 안건들의 **구현 방법론을 정리**하는 기획 문서 모음이다.

## TIPS 채택을 위한 핵심 관점
TIPS는 "기술 기반 창업"을 지원하므로, 단순 서비스 개선이 아니라 다음을 입증해야 한다.
1. **기술성**: 자체 기술/알고리즘의 난이도와 완성도.
2. **혁신성·차별성**: 기존에 없거나, 경쟁사가 쉽게 모방 못 하는 요소.
3. **R&D 요소**: 연구개발로 풀어야 하는 명확한 기술적 도전 과제.
4. **사업화 가능성**: 매출·확장·데이터 선순환 구조.

## 확정 전제 (사용자 제공)
- **개발 인력**: 충분하다고 전제(제약 아님).
- **예산**: 본 분석 결과에 연동해 책정 예정 → 비용 가이드(04) 제공.
- **보유 데이터**: 없음 → 인터넷 공개 데이터만으로 부트스트랩(03).
- **AI 엔진**: 최신 Claude 모델 API.

## 공통 설계 원칙
- **단계적·현실적**: 모든 시스템은 MVP → 확장 단계로 나눠 초기 비용·인력 부담을 낮춘다.
- **데이터 선순환**: 수집→분석→추천→입점사 실행→성과 데이터 재수집의 피드백 루프를 핵심 자산으로 본다.
- **AI 기본 엔진**: 최신 Claude 모델(Opus 4.8 / Sonnet 4.6 / Haiku 4.5) API를 기본 가정.
