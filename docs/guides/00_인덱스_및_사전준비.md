## MVP 최소 경로 (가장 빨리 도는 파이프라인)
`01 → 02 → 06 → 07 → 08`
→ 네이버 무료 API로 수집 → Postgres 저장 → Claude로 분류·인사이트 → cron으로 매일 자동 실행. (구글/유튜브/크롤링·스코어링은 그다음 단계)

## 비용 요약
- **02~05 데이터 수집**: 공식 무료 API/오픈소스 → **0원**(쿼터 내).
- **06 저장**: 로컬 Docker는 0원, 클라우드 관리형 DB는 월 수만 원.
- **07 LLM**: 사용량 과금. MVP 규모 월 십수만 원 (상세 `docs/04_비용가이드.md`).

## 보안 원칙 (모든 가이드 공통)
- API 키·시크릿은 **절대 코드/깃에 커밋하지 말 것**. `.env` + `.gitignore` 사용(→ 01).
- 키 노출 시 즉시 각 콘솔에서 재발급(rotate).
- 수집 시 각 서비스 **약관·robots.txt·rate limit 준수**(→ 05, 그리고 `docs/03` §5 체크리스트).
