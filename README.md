## 확정된 방향 (사용자 결정)
- **목표 규모**: 일반(기본) TIPS R&D **8억원** (스케일업·딥테크 등은 후속 확장 경로 참고).
- **운영사(액셀러레이터)**: 확보 완료 — TIPS 최대 게이트 통과.
- **문서 톤**: 기술 구현 설계 **+** TIPS 제안서 논리 **통합**.
- **현실 반영**: 단계적·현실적 (MVP → 확장).
- **AI 비용 분리**: 운영(API 종량, `05 §2~3`) ↔ 개발(Claude Code Max 20x, `05 §2.5`).
